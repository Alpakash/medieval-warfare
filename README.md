# Medieval Warfare #

<div align="left">

## :page_facing_up: Description

A React + TypeScript application where items can be put in the cart and bought with the user's gold. After purchase you can see your bought items in your inventory.

## How to install and run
```terminal
pnpm install
pnpm run dev
```

## Used Technologies
- React 19
- TypeScript (strict mode)
- Vite
- Tailwind CSS v4 + shadcn/ui
- Zustand
- react-router-dom v7
